# EE319K_Lab10

Inputs:
  1 slide pot
  1 shoot button
  2 directional buttons
  
Outputs:
  Sound
  LCD
  
Timers:
  Systick checks player input
  Timer0 tracks sound output
  Busy-Wait LCD
  
 Gameplay:
  static background
  2 humans. stand still
  2 enemies
  Lander:
    goes to a humans x coordinate, picks them up
  Mutant:
    moves randomly, tries to reach players y level and shoot them
